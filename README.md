## usage

```bash
# login
$ npm login --registry http://192.168.110.10:4873
# after login success
# install
$ npm install -g node-cli-temple --registry http://192.168.110.10:4873
# help
$ node-cli-temple --help
$ node-cli-temple --version
# uninstall
$ npm uninstall -g node-cli-temple
```
